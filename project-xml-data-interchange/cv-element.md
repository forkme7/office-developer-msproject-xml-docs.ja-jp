# CV 要素

CV はコスト差異とも呼ばれます。一般に、CV は、状況報告日または今日の日付までの期間で現行の完了レベルに達するために必要とされていたコストと、実際にかかったコストとの差異です。

** Task** の場合、** CV ** はタスクに適用されるコストの差異です。

** Resource** の場合、リソースに適用されるコストの差異です。

** Assignment** の場合、割り当てに適用されるコストの差異です。

    <CV>
      FloatValue
    </CV>
