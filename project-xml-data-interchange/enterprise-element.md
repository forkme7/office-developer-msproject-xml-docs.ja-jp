## 備考

Microsoft Office Project 2007 では、ローカル アウトライン コードをテキスト型のエンタープライズ ユーザー設定フィールドとして Microsoft Office Project Server 2007 に追加できます。このテキスト型のエンタープライズ ユーザー設定フィールドは、階層型参照テーブルを持ちます。
