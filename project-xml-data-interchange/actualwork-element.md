# ActualWork 要素

** Task** の場合、** ActualWork ** はタスクが割り当てられたすべてのリソースによって既に実行された作業時間です。

** Resource** の場合は、リソースに割り当てられたすべての割り当てについて既に実行された実績作業時間です。

** Assignment** の場合は、タスクについてリソースによって既に実行された作業時間です。

    <ActualWork>
      DurationValue
    </ActualWork>
