# OvertimeWork 要素

** Task** の場合、** OvertimeWork **は、タスクに割り当てられたすべてのリソースが行い、超過単価で課金される、超過作業の予定時間です。

** Resource** の場合は、リソースに割り当てられたすべてのタスクについて行われ、リソースの超過単価で課金される、超過作業の時間です。

** Assignment** の場合は、リソースがタスクについて行い、リソースの超過単価で課金される、超過作業の時間です。

    <OvertimeWork>
      DurationValue
    </OvertimeWork>
