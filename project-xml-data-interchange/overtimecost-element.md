# OvertimeCost 要素

** Task** の場合、** OvertimeCost ** はタスクの実績超過コストの合計です。

** Resource** の場合は、すべての割り当てタスクのリソースに関する合計超過コストです。

** Assignment** の場合は、リソース割り当ての合計超過コストです。

    <OvertimeCost>
      DecimalValue
    </OvertimeCost>
