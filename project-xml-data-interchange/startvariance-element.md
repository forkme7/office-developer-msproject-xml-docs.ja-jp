# StartVariance 要素

** Task** の場合、** StartVariance **はタスクの基準開始日と現在計画されている開始日の差異です。

** Assignment** の場合は、割り当ての基準開始日と現在計画されている開始日の差異です。

    <StartVariance>
      IntegerValue
    </StartVariance>
