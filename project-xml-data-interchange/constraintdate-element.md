# ConstraintDate 要素

タスクの** ConstraintType **で定義されている制限付きの開始日または終了日を示します。制約タイプが "できるだけ遅く" または "できるだけ早く" に設定されていない場合は必須です。

    <ConstraintDate>
      DateTimeValue
    </ConstraintDate>
