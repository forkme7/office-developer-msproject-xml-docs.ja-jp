# FinishVariance 要素

** Task** の場合、** FinishVariance ** は、タスクの基準終了日とタスクの現在の終了日との差を示す時間です。

** Assignment** の場合は、割り当ての基準終了日と割り当ての予定された終了日との差です。

    <FinishVariance>
      IntegerValue
    </FinishVariance>
