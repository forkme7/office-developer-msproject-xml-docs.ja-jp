# ID 要素

** Resource** の場合、** ID ** はリソース リスト内でのリソースの位置識別子です。

** Task** の場合は、タスク リスト内でのタスクの位置識別子です。

** Value** の場合は、プロジェクト内の値の ID です。

    <ID>
      IntegerValue
    </ID>
