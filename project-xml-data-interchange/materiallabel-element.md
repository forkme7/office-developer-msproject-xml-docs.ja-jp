# MaterialLabel 要素

数量単価型リソースに対して入力される、トン、箱数、立方ヤードなどの測定単位。数量単価型リソースの割り当ての** Units **と併せて使用し、リソースの** Type **が 1 (時間単価型リソース) に設定されている場合にのみ使用できます。

    <MaterialLabel>
      String(512)Value
    </MaterialLabel>
