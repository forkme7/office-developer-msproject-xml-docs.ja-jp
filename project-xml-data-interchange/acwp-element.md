# ACWP 要素

**ACWP** は、状況報告日または今日の日付までに実行された作業時間の実績コストです。

** Task** の場合、** ACWP ** は、タスクで既に終了した作業時間のコストです。

** Resource** の場合は、すべてのリソース割り当ての **ACWP** 値の合計です。

** Assignment** の場合は、リソースがタスクについて既に行った作業のコストです。

    <ACWP>
      FloatValue
    </ACWP>
