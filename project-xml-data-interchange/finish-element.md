# Finish 要素

** Task**、**Resource**、または **Assignment** 要素の場合、** Finish **はタスクの完了予定の日時です。

** Baseline** の場合は、ベースラインが保存されたときの、タスク、リソース、または割り当ての終了予定日です。

** TimephasedData** の場合は、タスク、リソース、割り当て、またはベースライン時間配分データ期間の終了日です。

    <Finish>
      DateTimeValue
    </Finish>
