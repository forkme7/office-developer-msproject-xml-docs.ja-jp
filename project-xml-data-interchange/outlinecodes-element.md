## 備考

アウトライン コードは、ローカル プロジェクトのタスクまたはリソースに関連付けることができます。階層型参照テーブルを持つエンタープライズ テキスト ユーザー設定フィールドとして、ローカル アウトライン コードを Microsoft Office Project Server 2007 に追加できます。
