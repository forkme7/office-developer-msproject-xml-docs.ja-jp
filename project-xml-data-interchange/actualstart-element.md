# ActualStart 要素

** Task** の場合は、** ActualStart ** はタスクが実際に始まった日付と時刻です。

** Assignment** の場合は、割り当てが実際に始まった日付と時刻です。

    <ActualStart>
      DateTimeValue
    </ActualStart>
