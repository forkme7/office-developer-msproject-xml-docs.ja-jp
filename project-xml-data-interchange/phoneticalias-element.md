# PhoneticAlias 要素

ひらがなまたはカタカナのいずれかで示したユーザー設定フィールドまたはアウトライン コードの読み方です。Microsoft Office Project の日本語版でのみ使用します。

    <PhoneticAlias>
      StringValue
    </PhoneticAlias>
