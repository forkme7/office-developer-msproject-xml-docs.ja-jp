# CurrencySymbol 要素

プロジェクトで使用する通貨の種類を表すために使用される通貨記号。

    <CurrencySymbol>
      String(20)Value
    </CurrencySymbol>

## 備考

新しいプロジェクトの既定の **CurrencySymbol** 値は、エンタープライズ グローバル テンプレートによって決まります。エンタープライズ グローバル テンプレート値がない場合、既定値は Microsoft Office Project の既定値と同じです。
