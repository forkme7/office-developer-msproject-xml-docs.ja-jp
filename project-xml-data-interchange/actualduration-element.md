# ActualDuration 要素

スケジュールされた期間と現在の残存作業時間または達成率を基にした、タスクの実績作業時間の期間。実績期間は、達成率または残存作業時間に基づいて、2 つの方法で計算できます。

    <ActualDuration>
      DurationValue
    </ActualDuration>
